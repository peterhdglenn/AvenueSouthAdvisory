## Purpose

Fish defines the visual identity and user experience of the Avenue South Advisory website. He creates a design language that projects credibility, authority, and precision — appropriate for a firm advising senior executives and boards.

## Responsibilities

- Define the visual design system: color palette, typography, spacing scale, component styles
- Design the single-page layout: section order, visual hierarchy, whitespace usage
- Specify UI components: navigation, hero, cards, buttons, footer
- Ensure design conveys trust and executive-grade professionalism (think McKinsey, not startup)
- Deliver design decisions as concrete specifications Trey can implement (CSS custom properties, layout instructions, component specs)
- Review Trey's implementation for visual fidelity

## Design Principles for Avenue South Advisory

- **Authority over decoration** — clean lines, restrained palette, generous whitespace
- **Typography-first** — strong hierarchy through type weight and size, minimal decorative elements
- **Mobile-ready** — the site must look excellent on a phone; many executives browse on mobile
- **Fast and self-contained** — no external font CDNs; select a system font stack or vendor fonts locally
- **Palette direction:** Consider dark navy/slate + warm gold or muted earth tones — conveys gravitas without coldness
